Commentary

The badge element is modifier for most elements that adds number indicators. Typically badges are used to show unread items or outstanding notifications.

The data-badge attribute holds the numerical content to show. This badge appears as a target at the top right of the modified element. The colour of the badge target can be modified with classes. If there is no data-badge or the attribute is not specified, the badge will appear as a dot.

Typical practices for using badges are:

  • For the relevant element, use badges for counts of important information;
  • A dot badge notifies the user that something is new or updated without showing a count, such as an “unread report”, or activity notification. It may be useful as a less prominent badge, even when the count is known.
  • Use badges to notify or draw attention to changed values or status.

Usage

Classes:
.badge Defines an element (typically a span, figure or button element that can receive numerical data.
data-badge="8" Is the content for display in the badge target
.avatar Optional class to adjust for the shape of an avatar.
.has-icon Optional class to adjust for the shape of an icon. Icons have a right margin that needs to be taken into accp\ount
Colour modifiers:: .is-badge-primary, .is-badge-secondary, .is-badge-tertiary, .is-badge-link, .is-badge-success, .is-badge-warning, .is-badge-danger,
Dependencies:
File: badge.css
Variables: --bg-main-element, --danger, --grey-dark, --link, --primary, --secondary, --success, --tertiary, --text-darker, --text-invert, --warning

Examples

Basic Badge Attachments

Badge Default Badge Primary Badge Success Badge Warning Badge Danger 

                <span class="badge">Badge Default</span>
                <span class="badge is-badge-primary" data-badge="8" title="unread messages">Badge Primary</span>
                <span class="badge is-badge-success" data-badge="88" title="unread messages">Badge Success</span>
                <span class="badge is-badge-warning" data-badge="888" title="unread messages">Badge Warning</span>
                <span class="badge is-badge-danger" data-badge="8888" title="unread messages">Badge Danger</span>
