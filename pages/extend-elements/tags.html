Commentary

The tag element is adapted from Bulma. The difference is that Goldilocks-CSS uses the native HTML data tag with value attribute set to content text. This allows screen readers to extract relevant semantic information.

The tag element provides add on information for a variety of use cases. The tag elements can be concatenated into a tracking list with target click areas.

Usage

Classes:
.tag Defines tag and its setup.
.is-rounded Optional modifier rounds the tag.
.tags Creates horizontal grouped tags using flexbox.
.has-addons Combines tags into one display component.
Size options: .tag-normal, .tag-medium, .tag-large
Colour options: Any of the element colour modifiers from the Colour Palette helpers.
Dependencies:
File: tag.css
Variables: --mar-horz, --mar-vert

Compound Tags


                <div class="tags are-medium">
                    <data class="tag" value="All"></data>
                    <data class="tag" value="Medium"></data>
                    <data class="tag" value="Size"></data>
                </div>

                <div class="tags are-large">
                    <data class="tag is-success" value="All"></data>

                </div>

                <data class="tag is-success" value="Success + Delete">
                    <button class="delete is-small" aria-label="delete"></button>

                    <button class="delete is-large" aria-label="delete"></button>
