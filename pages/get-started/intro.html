A customer first CSS design library.

Goldilocks-CSS is a lightweight, responsive, standards compliant and up-to-date HTML5 / CSS3 design library to meet the needs of product / business managers who need to pay for and manage scarce resources.

Key Points

Customer

Fully responsive across any device

Selectable base font size

Selectable light / dark mode

Fast time to first paint - small footprint cached

Faster render - reduced number of HTML tags

Business

Stay out of legal quagmire - meets WebAim accessibility requirements

Future proofed - HTML5 and CSS3 Standards compliant

Lower cost - no javascript means less code

Simplify maintenance - expected to be THE source of truth

Technology

No breakpoints - uses design widths

Simplified customization with CSS variables

No third party libraries

No dependencies and no bundler required

Supports triadic colour scheme
