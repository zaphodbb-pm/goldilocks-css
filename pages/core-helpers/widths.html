Commentary

These are common helpers that force a width on an element. These are useful to provide controls for images and other elements that do not have an intrinsic size.

Usage

Classes:
Fixed widths .has-width-1rem, .has-width-2rem, .has-width-3rem, .has-width-4rem, .has-width-5rem, .has-width-6rem, .has-width-7rem, .has-width-8rem, .has-width-10rem, .has-width-15rem, .has-width-20rem, .has-width-25rem, .has-width-30rem, .has-width-40rem
Relative % widths .width-25, .width-33, .width-50, .width-75, .width-full
