Commentary

Breadcrumbs are simple navigational hierarchies to indicate the path to the current page location. The breadcrumb navigation component uses a nav HTML tag containing a ul list.

The link separator are created in the li::before pseudo-element. Four separator types are supported. Three size options and two positioning options are also available

Usage

Classes:
.breadcrumb Defines main container for breadcrumb, typically nav html tag is used at top level.
Positioning options: .is-centered, .is-right
Size options: .is-small, .is-medium, .is-large
Separator options: .has-arrow-separator, .has-bullet-separator, .has-quote-separator
Dependencies:
File: breadcrumb.css
Variables: --breadcrumb-separator, --link, --pad-horz, --size-4, --size-5, --size-7, --text-darker
