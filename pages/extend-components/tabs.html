Commentary

Tabs is an all CSS component (no javascript required) that that provides basic responsive horizontal content tabs with two different styles. Tabs allows for the addition of icons in the tab label. Note that a maximum of 9 tab labels are supported. More than 9 tabs makes the element cluttered and hard for the user to comprehend. Break a long list of items into two tabbed sections or combine some items together if you can.

Tabs maintains the semantic HTML structure for screen readers of the user relevant content. This requires a semantic structure for the tab-content area and includes a section tag and an h2 heading. The heading is off-screen but still readable by a screen reader.

Based on work by Jason Knight

Usage

Classes:
.tabs Defines main container with default tab layout.
.is-boxed Provides an alternative layout with classic tab border.
.tabs-centered, .tabs-right Optional modifiers to display tabs in center or right adjusted.
.is-fullwidth Optional modifier that spreads tab labels across the full width of the element.
.is-small, .is-medium, .is-large Optional modifiers to change tab label text size.
Dependencies:
File: tabs.css
Variables: --bg-main, --bg-main-invert, --bg-main-ter, --border-main-darker, --link, --padding, --radius --size-4, --size-5, --size-7, --text, --text-darker
